% dotfiles

My dotfiles managed by [dotdrop](https://github.com/deadc0de6/dotdrop).

# Installation
```bash
git clone https://github.com/cangermueller/dotfiles.git`
cd dotfiles
bash ./dotdrop.sh install -p base
```

Installing [fzf](https://github.com/junegunn/fzf):
```bash
cd dotfiles/fzf
./install
```
